Free Symbol and Character Info: Count and Decode Unicode plus ASCII with Names and HTML Entities

Paste text to inspect every symbol. Get Unicode code points, official style names, scripts and blocks, UTF-8 and UTF-16, ASCII values, and copy ready HTML entities. Great for writers, developers, designers, and QA.

Symbols can look the same but behave differently across apps. Accents, emoji skin tones, flags, and zero width joiners can break counts, layout, or encoding. The SnipText Symbol and Character Info gives instant clarity with accurate grapheme counts, Unicode and ASCII details, names, blocks, scripts, and HTML entities so you can paste, inspect, and ship with confidence.

What the tool shows

  • Grapheme cluster counts: Treats emoji sequences, flags, and combining marks as one user seen character.
  • Unicode and ASCII: Code points like U+1F600, ASCII codes when available, and plane info.
  • Names, scripts, blocks: Human readable names with Unicode block and script hints.
  • Encodings: Exact UTF-8 bytes and UTF-16 code units for debugging and integrations.
  • HTML entities: Copy named or numeric entities for safe web output.
  • Unique symbol summary: Frequency and dedupe views to spot odd or repeated glyphs fast.

How to use it fast

  1. Open Symbol and Character Info.
  2. Paste or type your text. Tiles appear for each unique symbol with counts.
  3. Click a tile to see Unicode, ASCII, names, scripts and blocks, UTF-8 and UTF-16, and HTML entities.
  4. Use Copy Report to export a full breakdown for tickets, docs, or QA notes.
  • Grapheme vs code points: Enable Count grapheme clusters to match what users perceive as one character.

Advanced options you will love

  • Visibility controls: Show invisibles, exclude whitespace from counts, switch grapheme or code point modes.
  • Transform tools: Case convert, URL encode or decode, HTML escape or unescape, unique only view, and sorting.
  • Favorites and quick picker: Pin frequent glyphs and re insert them in one click. Export or import sets.
  • Random samplers: One click sets like arrows, currency, math, ASCII punctuation, emoji, and Latin 1 for testing.
